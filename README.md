ProjectV7.py and myfooddatabaseV4.db are the latest versions.
*(Please make sure you have installed all the libraries, the V4 database and have used the correct API key before you run)*
*It is suggested to run on the terminal

Libraries used are (for reference and installation):
1.tkinter
2.sqlite3
3.pyttsx3
4.time
5.datetime
6.dateutil
7.cv2

Older versions can be viewed from the second branch.

The API key is owned by Jaydus Poon, please send an email to  poonjaydus@gmail.com for permission to use.
Please bear in mind that the API key only gives access the object detection, the other functions can be still used even without the API.
